## Как начать работу с кодом


### В терминале Git Bash
```sh
$git clone "https://github.com/14xyegriz88/WebstoreDOG.git"
```
### В терминале Powershell
```sh
cd WebstoreDOG
```
![Image alt](https://github.com/14xyegriz88/WebstoreDOG/blob/main/image.png)
```sh
npm run dev 
```
